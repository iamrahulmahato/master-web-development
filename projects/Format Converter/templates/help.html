How to Use This Application


Step-by-Step Guide

  1. Navigate to the Edit Page: Click on the "Edit Image" link in the navigation bar to access the image editing feature.
  2. Select an Image: On the edit page, you will find an option to upload an image. Click on the "Choose File" button to select an image from your computer. Supported formats include PNG, JPG, JPEG, GIF, and WEBP.
  3. Choose an Operation: Below the file selection, you will see a dropdown menu where you can choose one of the following operations:
    • Convert to Grayscale (cgray): Converts the image to grayscale.
    • Convert to WebP (cwebp): Converts the image to WebP format.
    • Convert to JPG (cjpg): Converts the image to JPG format.
    • Convert to PNG (cpng): Converts the image to PNG format.
  4. Upload and Process: Once you have selected the image and chosen an operation, click on the "Upload" button. The application will process your image based on the selected operation.
  5. Download the Processed Image: After the image is processed, a message will appear with a link to the processed image. Click on the link to view the image, and right-click to download it if needed.

Troubleshooting

  • No File Selected: If you see an error saying "No file part" or "No selected file", please make sure you have chosen an image before clicking "Upload".
  • Unsupported File Format: Only the following formats are supported: PNG, JPG, JPEG, GIF, WEBP. Ensure the file you are uploading is in one of these formats.
  • Processed Image Not Showing: If the link to the processed image is not working, please refresh the page and try again.
